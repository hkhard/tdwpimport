switch()The switch() function works similarly to the if() function. It takes a variable number of parameters. The first parameter is a conditional value. Further parameters should be provided in pairs. The first parameter of each pair is a comparison value, and the second parameter of each pair is a result value.

The switch() function compares the conditional value with the first (comparison) value of each pair of parameters following it. When it locates a matching value, it returns the result parameter of that matching comparison value.

switch(3, 1, 10, 2, 20, 3, 30, 4, 40)
In this example, the conditional parameter is the first parameter (3). The following parameters are pairs of comparison and result values: 1 and 10, 2 and 20, 3 and 30, and 4 and 40. The conditional parameter is compared to each of the comparison parameters (1, 2, 3, and 4). When a match is found (3), the pair's result parameter (30) is returned. Therefore, in this example, 30 would be returned.

If no match is found, 0 is returned. A final parameter may be supplied if you wish a default value (of something other than 0) to be returned when no matching comparison value is found.

switch(6, 1, 10, 2, 20, 3, 30, 4, 40, -1)
In this example, the conditional parameter (6) doesn't match any of the comparison parameters (1, 2, 3, 4). Therefore, the final (default) parameter is returned, -1.

Of course, in practice it doesn't make sense to provide a literal number as the conditional (first) parameter. Instead, a variable is usually used. For example, switch(rank, 1, 100, 2, 70, 3, 40) returns 100 when the rank variable is 1, 70 when the rank variable is 2, and 40 when the rank variable is 3. This could be used to provide additional bonus points to the top 3 ranks.

lswitch()The lswitch() function (linear switch) is a simpler version of the switch() function. Instead of taking pairs of values, it assumes the first comparison value is always 1 (one), the second is 2 (two), and so on. Therefore, only the result value of each value pair needs to be provided. The final parameter plays the additional role of being the default value, for any comparison values that are not included in the parameter list.lswitch(1, 10, 20, 30, 40)
In this example, the conditional parameter is the first parameter (1). The following parameters are all result values: 10, 20, 30, and 40. The conditional parameter specifies which result value is chosen. In this case, the conditional value is 1, and therefore the first result value is chosen (10).

lswitch(4, 10, 20, 30, 40)
In this example, the conditional parameter is 4. Therefore the 4th result value is chosen (40).

lswitch(10, 10, 20, 30, 40)
In this example, the conditional parameter is 10. However, there are only 4 parameters following the conditional parameter. There is no 10th parameter. The lswitch() function assumes the final parameter is the default parameter. Therefore the final parameter is chosen (40).
